== Description ==

Vertex Addons For Elementor Builder is designed to enhance your page-building experience like never before. It provides a diverse range of powerful and feature-packed widgets that integrate seamlessly with Elementor. Whether you're building a complex website or a simple landing page, Vertex offers all the tools you need to create stunning web designs without writing a single line of code.

== Installation ==
*  Upload the vertex-addons.zip file to the /wp-content/plugins/ directory, or install the plugin through the WordPress Plugins screen directly.
*  Activate the plugin through the ‘Plugins’ screen in WordPress.
*  Navigate to Elementor > Vertex Addons to explore all widgets.
